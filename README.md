# 仿写助手

一个基于 Claude API 的智能文案仿写工具，帮助你快速生成风格相似的微头条文案。

## 🚀 在线体验

访问：[https://你的用户名.github.io/仓库名/](https://你的用户名.github.io/仓库名/)

## ✨ 功能特点

- 🎯 **智能仿写**：基于 Claude 模型，生成高质量仿写文案
- 🔧 **灵活配置**：支持自定义主题、生成数量和风格偏好
- 📱 **响应式设计**：完美适配手机和电脑
- 🔒 **隐私保护**：API Key 仅存储在本地浏览器中
- 📋 **一键复制**：快速复制生成的文案内容

## 🛠️ 使用方法

1. **获取 API Key**
   - 访问 [LongCat API 控制台](https://api.longcat.chat)
   - 注册账号并获取你的 API Key

2. **使用工具**
   - 在 API Key 输入框中填入你的 LongCat API Key
   - 在原始文案区域粘贴你想要仿写的微头条内容
   - （可选）展开高级设置，自定义仿写参数
   - 点击"生成仿写文案"按钮
   - 复制生成的内容使用

## ⚙️ 高级设置

- **仿写主题**：指定仿写内容的主题方向
- **生成数量**：选择生成 1-3 篇不同版本
- **风格偏好**：
  - 与原文一致：保持原文风格
  - 更口语化：使用更通俗的表达
  - 更正式：采用更规范的用语

## 🔒 隐私说明

- API Key 仅存储在你自己的浏览器中
- 不会上传到任何第三方服务器
- 所有数据都在本地处理
